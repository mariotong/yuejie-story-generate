# 岳姐专门口播总结：30-60秒固定机位口播

用于用户要“专门给岳姐口播”“一分钟口播”“10s一个镜头”“口播故事版/分镜提示稿”时。

## 核心判断

岳姐口播不是剧情分镜，也不是旁白复述。它应该像岳姐坐在镜头前，借故事说一个中年女人的现实处境。

优先形式：

- 30秒：3个10秒镜头。
- 60秒：6个10秒镜头。
- 固定机位或轻微推进；不要切家庭剧情画面。
- 全片可以只用一张岳姐正面半身参考图；如果用户追求更自然，最多2张（正面+轻微低眼/侧身），不要给太多角度，避免人物漂移。

## 参考图策略

最低配置：

```text
@image1 是岳姐正面半身参考图：成熟中国女性，短发，素色针织衫，坐在温暖明亮的会客厅里，正面面对镜头，真实生活化，不年轻化，不网红脸。
全片严格参考 @image1：同一个人、同一张脸、同一发型、同一服装、同一会客厅、同一种自然光。
固定机位口播，不切换角度，不换场景。
```

更稳配置：

```text
@image1：岳姐正面半身，看镜头。
@image2：同场景、同服装，岳姐轻微低眼或说完停顿。
```

不要准备太多角度。角度越多，图生视频越容易换脸、换年龄、换气质。

## 口播稿写法

用户明确反馈过：

- 不要突然引入前文没铺过的词，如前面没有“我来吧”，结尾就不要硬说“我来吧”。
- 不要把口播写成剧情复述，比如“她一住院，全家才发现，原来谁都没学会生活”会显得假、莫名其妙。
- 岳姐口播要像对屏幕前的人说现实话：观点清楚、生活细节足、不过度煽情、不喊口号。
- 口播提示词要像“真人口播”，不要像复杂导演分镜。分镜控制可以写，但核心必须是“面对镜头连续自然说完一段话”。

好口播结构：

```text
1. 先说普遍处境：很多女人不是被大事压垮，而是每天小事都找她。
2. 列具体小事：饭几点吃、药放在哪、孩子的事谁记着、老人哪天复查、家里缺什么。
3. 算长期消耗：单件事不重，年年月月压一个人身上，人会被压空。
4. 点破看不见：做久了别人就看不见她在记、在想、也会累。
5. 扎心观点：最消耗的不是没人帮忙，是所有人都默认你会处理、不会倒下。
6. 收束边界：人不是机器，家里的事别只等一个人兜底。
```

## 60秒推荐口播稿

```text
很多女人真正累的，
不是家里出了什么大事。

是每天那些没人当回事的小事，
都在找她。

饭几点吃，药放在哪，
孩子的事谁记着，
老人哪天复查，
家里缺什么东西。

这些事单拎出来，
都不算重。

可它们年年月月压在一个人身上，
人是会被压空的。

更难受的是，
她做久了，
别人就看不见了。

看不见她在记，
看不见她在想，
看不见她其实也会烦、会累、会撑不住。

一个家最消耗人的，
不是没人帮忙。

是所有人都觉得：
你会记得，
你会处理，
你不会倒下。

可人不是机器。

你不能因为一个人一直能扛，
就默认她应该一直扛。

家里的事，
谁都别只等一个人兜底。

能问一句，
能搭把手，
能自己记住一件事，

这个家，
就不至于把一个人耗干。
```

## 60秒拆镜头

```text
镜头1｜0-10秒｜小事才最耗人
很多女人真正累的，不是家里出了什么大事。是每天那些没人当回事的小事，都在找她。

镜头2｜10-20秒｜没人看见的小事
饭几点吃，药放在哪，孩子的事谁记着，老人哪天复查，家里缺什么东西。

镜头3｜20-30秒｜小事压久了会空
这些事单拎出来，都不算重。可它们年年月月压在一个人身上，人是会被压空的。

镜头4｜30-40秒｜做久了就被看不见
更难受的是，她做久了，别人就看不见了。看不见她在记，看不见她在想，看不见她其实也会累。

镜头5｜40-50秒｜默认她不会倒
一个家最消耗人的，不是没人帮忙。是所有人都觉得：你会记得，你会处理，你不会倒下。

镜头6｜50-60秒｜别只等一个人兜底
可人不是机器。你不能因为一个人一直能扛，就默认她应该一直扛。家里的事，谁都别只等一个人兜底。
```

## 口播视频提示词模板

```text
镜头N｜10秒｜岳姐面对镜头口播：{本段主题}

参考图：
使用同一张 @image1 岳姐参考图。
必须保持岳姐同一张脸、同一发型、同一服装、同一会客厅。

画面：
岳姐坐在温暖明亮的会客厅里，正面半身面对镜头。
固定机位真实口播，最多轻微自然呼吸感或轻微推进。
不要切镜头，不要换角度，不要插入家庭画面，不要字幕，不要文字，不要水印。

口播台词：
“{本镜头完整口播台词}”

口播方式：
这是岳姐本人对着观众连续自然说话，不是旁白，不是剧情对白。
全程只有岳姐一个人出镜，只有岳姐嘴唇动。
语气平稳、直接、清醒，像坐下来跟屏幕前的姐妹说话。
不要演哭，不要激动，不要皱眉过度，不要大幅挥手。

声音控制：
只有岳姐本人说话。不要旁白，不要画外音，不要背景人物声音。
嘴型尽量匹配中文口播。

情绪：
{本段真正的情绪任务，例如：平静开场 / 列出隐形负担 / 点破理所当然 / 清醒收束}
```

## 故事版图片注意

如果用户要“根据这几个口播镜头生成故事版”，可以生成一张9:16中文表格图：标题 + 基础信息 + 6行镜头表 + 底部参考图策略。

图上文字不要堆太多；如果生成图中文字不清晰，下一版要减少列和字数，只保留：镜头、主题、口播、画面、情绪。不要继续堆完整提示词到图片里。
