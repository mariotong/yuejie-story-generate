# 给其他 Agent 的故事版复现提示词

当用户不是要直接生成故事版图片，而是要一段可复制给其他 agent / 其他图片模型使用的提示词时，使用本文件。

常见口令：

```text
给我一个生成故事版的提示词给其他agent使用。
给我一段prompt，让别的agent复现你这个故事版图。
我想把这个故事版交给别的AI生成，提示词怎么写？
```

## 输出原则

- 只输出可复制的完整提示词，不直接生成图片。
- 提示词要能复现“完整故事版图片”，不是单独镜头图。
- 必须包含：画布比例、标题区、元信息区、表格列、9个镜头、底部字幕、视觉风格、禁止项。
- 如果用户给了文章，要把文章核心信息压进提示词里；不要要求其他 agent 再读长文。
- 语言默认中文。

## 复现提示词模板

```text
请生成一张竖版中文短视频故事版图片，比例 9:16，整体像专业分镜表/故事版海报。

视觉风格：
白色偏暖底，浅橙色表格线，黑色正文，重点字用橙红色。整体明亮、干净、清楚，适合中老年手机观看。不要灰暗，不要苦情海报感，不要花哨设计。

顶部标题：
{故事标题}

标题下方元信息：
总时长：约{总秒数}秒｜{镜头数}个镜头｜每个镜头{单镜头秒数范围}秒｜画面比例：9:16竖屏｜风格：真实生活化/明亮清楚/中老年爱看

主体表格：
做成9行分镜表，每行一个镜头。表格列从左到右：
镜头、画面内容、首帧、尾帧、时长、时间轴：台词/旁白、转场/情绪承接、其他：场景/细节、image2生成策略。

每一行的“首帧”和“尾帧”都要有一张小图，真实生活化照片风格，明亮自然光，人物脸部清楚可见，不要文字水印。

镜头内容：
1. {镜头1内容，含首帧、尾帧、时长}
2. {镜头2内容，含首帧、尾帧、时长}
3. {镜头3内容，含首帧、尾帧、时长}
4. {镜头4内容，含首帧、尾帧、时长}
5. {镜头5内容，含首帧、尾帧、时长}
6. {镜头6内容，含首帧、尾帧、时长}
7. {镜头7内容，含首帧、尾帧、时长}
8. {镜头8内容，含首帧、尾帧、时长}
9. {镜头9内容，含首帧、尾帧、时长}

底部结尾字幕：
{结尾金句}

底部小字：
转场建议：物件转场+声音延续切｜背景音乐：温暖、克制、生活纪录片感

重要要求：
- 单个镜头不能超过10秒。
- 整体要像一张完整可读的故事版表，不是九宫格海报。
- 所有小图都要明亮清楚，真实生活化，中老年观众爱看。
- 不要暗黑色调，不要强滤镜，不要广告感，不要夸张表演。
- 表格里的中文尽量清晰可读。
- 小图里不要出现文字、字幕、水印。
```

## 基于文章的填充方式

如果用户给的是文章，先填充这些变量：

- `{故事标题}`：从文章提炼冲突+观点反转。
- `{总秒数}`：通常 72-78 秒。
- `{镜头数}`：通常 9 个。
- `{单镜头秒数范围}`：8-9 秒，且每个镜头不超过 10 秒。
- `{镜头1-9内容}`：用可视化动作和场景写，不照搬文章段落。
- `{结尾金句}`：用文章里最能转发的一句，或压缩成一句。

## 林姐文章示例标题和结尾

```text
故事标题：
女人别再说“我来吧”，这不是贤惠，是把自己放到全家最低成本

结尾金句：
别再轻易说“我来吧”。说一次是帮忙，说十年就是消耗。
```
