# 单张人物参考图 → 6秒口播分镜提示词经验

适用场景：用户发一张岳姐/中年女性人物图，问“这张适合啥”“直接给提示词”“家庭关系口播”“分镜口播提示词”等。

## 用户偏好

- 用户通常要的是可直接复制到 Grok/图生视频的提示词，不要长篇解释。
- 如果用户说“直接给我提示词”，只输出一个完整代码块；不要再分析适合方向。
- 如果用户说“适合啥，直接给分镜口播提示词”，先用 1 句点明方向，再给完整提示词。
- 口播时长常为 6 秒，台词必须短：三段各 6-14 字左右，避免 6 秒塞长金句。

## 画面判断与选题匹配

1. 中式会客厅、桌前坐姿、盆景/格栅/青瓷/暖光：
   - 最适合“岳姐会客厅式家庭关系口播”。
   - 主题：中年女人别全家兜底、家里最累的人、边界感、情绪松弛。
   - 示例口播：
     “女人到中年， / 别什么事都替家里人兜着。 / 你兜得越多，他们越觉得理所当然。”

2. 玉石文玩店、柜台、玉雕、人物沉稳正坐：
   - 适合“借玉说人”的中年女性清醒口播。
   - 主题：稳住气、不炫富、不玄学、见过事后的分寸感。
   - 示例口播：
     “女人到了一定年纪， / 戴什么玉不重要， / 别再让自己乱了气。”

3. 玉石直播间、手机支架、满桌玉器、人物正在讲解：
   - 适合“直播间里借玉讲女人清醒”。
   - 主题：不硬卖货，借玉讲女人别亏待自己、给自己留体面。
   - 示例口播：
     “女人买玉， / 买的不是贵， / 是提醒自己，别再活得太亏。”
   - 家庭关系替换版：
     “女人到中年， / 别只顾着给家里花钱， / 也要舍得给自己留点体面。”

## 提示词结构

直接输出一个可复制代码块，包含：

```text
9:16竖屏真实口播视频。参考图中{人物+场景+服装+关键道具}。

视频必须保持参考图里的人物、服装、发型、表情、道具、背景和灯光不变。不要换人，不要换衣服，不要换场景，不要新增人物。

全程只有这位女性一个人出镜并口播。只有她本人嘴唇自然微动，不要旁白，不要画外音，不要字幕，不要水印，不要可读文字。

镜头从{中景/中近景}开始，{手部/玉器/桌面/背景}清楚可见。镜头缓慢稳定推进到半近景，突出她温和但有分量的眼神。整体真实自然，不要广告感，不要表演腔，不要网红脸，不要磨皮过度。

0.0s-1.0s：
{安静看向镜头/整理玉珠/双手交叠}，先不说话。

1.0s-2.4s：
她说：“{第一短句}”

2.4s-4.0s：
她说：“{第二短句}”

4.0s-6.0s：
她说：“{第三短句}”

画面情绪：
不是{错误方向1}，不是{错误方向2}，而是{正确情绪}。

最终要求：
保持同一人物、同一服装、同一场景、同一光线。不要出现字幕、水印、文字、招牌、手机屏幕内容或无关人物。
```

## 注意

- 对直播间玉器图，必须强调“不是硬卖货/不是叫卖/不是炫富”。
- 对家庭关系图，必须强调“不是抱怨/不是吵架/不是控诉”，而是平静提醒。
- 对玉石文玩图，必须强调“不是玄学养生/不是卖玉”，而是借物说人。
- 如果用户只要提示词，不要给“备选口播句子”“我建议”等额外内容。
