# Grok 短时长图生视频拆分模式

用于岳姐故事版在 Grok 额度/时长不足时，把原 8-10 秒镜头拆成 5 秒左右的 A/B 子镜头。目标是降低图生视频执行难度：每个子镜头只做一个动作变化、一个情绪落点。

## 总原则

- 命名：`镜头{N}A`、`镜头{N}B`。用户说“分镜头1/第一个分镜头/{N}a”输出 A；说“分镜头2/{N}b”输出 B。
- 每个 5 秒子镜头仍必须完整写：`@image1`、`@image2`、人物一致性、旁白主导规则、首尾图关系、镜头运动、逐秒画面、最终要求、合理性审查。
- 每个子镜头只承担一个任务：不要在 5 秒里塞多个人物入镜、多个对白和复杂走位。
- A 的尾帧要能自然接 B 的首帧；B 的尾帧落到原镜头的情绪终点。
- 旁白优先，人物少开口。旁白时画面人物必须闭嘴，不要对口型。
- 如旁白超过 16 字，优先压短句子，不要靠加快语速解决。

## 5 秒推荐骨架

```text
0.0s-1.5s
画面：从 @image1 开始，建立动作和场景。环境声，不要对白。

1.5s-3.2s
画面：核心动作变化。
旁白精确出现：
1.6s-3.1s
旁白说：
“{一句11-16字的故事解释}”

3.2s-5.0s
画面：自然接近 @image2，完成一个明确情绪或动作落点。尽量不要新增对白。
```

如果必须放人物声音：

```text
声音精确出现：
{start}s-{end}s
{画外音/人物名}说：
“{短句}”

说话人控制：
画外音人物不要入镜。入镜人物开口最多一次；旁白时必须闭嘴。
```

## 已确认的拆分范式

### 镜头1：体检单 + 被接住

- `1A`：林姐攥着体检单低头沉默 → 把体检单放到桌上，说“我撑不住了”。
  - 旁白：“她不是突然撑不住，是撑太久了。”
  - 林姐短对白：“我撑不住了。”
- `1B`：岳姐递纸巾 → 林姐终于掉泪。
  - 旁白：“她说完这句，就低下了头。” / “这一刻，她才敢掉眼泪。”
  - 岳姐短对白：“慢慢说。”

### 镜头2：厨房被打断

- `2A`：林姐切菜 → 手机亮 → 第一次被打断。
  - 旁白：“她一进厨房，就像进了全家的后台。”
  - 画外音：“妈，签字！”
  - 林姐不说话，只停一下。
- `2B`：林姐继续做饭 → 低声回应 → 拿锅铲回头忍住。
  - 林姐短对白：“放桌上。”
  - 旁白：“她手没停，心也没停过。” / “没人问她累，只问东西在哪。”

### 镜头3：餐桌琐事 + 家人等待

- `3A`：餐桌琐事摊开 → 桌面更满。
  - 旁白：“一桌小事，都落到她手里。”
  - 不安排人物对白。
- `3B`：丈夫孩子站旁边等 → 林姐继续处理。
  - 画外音可模糊：“这个你看一下。”
  - 旁白：“不是她记性好，是全家都懒得记。”

### 镜头5：医院警报

- `5A`：半夜胃疼进医院 → 坐在病床边发怔。
  - 旁白：“那天半夜，她胃疼到进了医院。”
  - 不安排人物对白，不出现医生护士家人。
- `5B`：医生看检查单 → 林姐按着胃发怔。
  - 旁白最稳版：“医生看单子，她看见的是亏空。”
  - 医生和林姐都不说话，避免对口型。

### 镜头6：住院后手机还在找她

- `6A`：林姐输液刚安静 → 手机又亮起。
  - 旁白：“她躺下了，家里的事却没停。”
  - 不安排人物对白。
- `6B`：手机里又有人找她 → 她无力看着手机不接。
  - 手机画外音：“妈，在哪儿？”
  - 旁白：“她住院了，家里还在找她。”

### 镜头7：岳姐点破核心

- `7A`：岳姐说破核心问题 → 林姐沉默被击中。
  - 旁白：“岳姐看出来，她不是累一天。”
  - 岳姐短对白稳版：“你把自己，活成了工具。”
- `7B` 可承接：林姐被说中后的沉默/重新理解自己，把情绪留给后续行动镜头。

## 审查要点

每次输出后必须给“合理性审查”，至少检查：

1. 子镜头任务是否单一。
2. 旁白/对白时长是否合理。
3. 说话人是否清楚，旁白时人物是否闭嘴。
4. @image1 到 @image2 的动作变化是否连续。
5. 和相邻 A/B 镜头是否能衔接。
