# 故事版格式和镜头规则

## 参考图结构

参考图是一张完整的短视频故事版表：

- 顶部大标题：情绪冲突钩子 + 价值反转。
- 基础信息：总时长、镜头数、每个镜头时长、画面比例、整体风格。
- 主体表格：每一行是一个镜头。
- 底部信息：结尾字幕、转场建议、背景音乐建议。

参考图大约 72 秒，9 个镜头，每个镜头 6-10 秒，9:16 竖屏，风格是真实、生活化、中老年爱看。受 Grok 限制，单个镜头不能超过 10 秒。

## 表格字段

| 字段 | 作用 | 写法 |
| --- | --- | --- |
| 镜头 | 镜头编号 | 1-10，通常 6-10 个镜头。 |
| 画面内容 | 剧情动作 | 一到两句话，写清楚动作和情绪含义。 |
| 首帧 image2 提示词 | 镜头开始画面 | 描述这个镜头开头要生成的画面。 |
| 尾帧 image2 提示词 | 镜头结束画面 | 描述这个镜头结尾的情绪变化或动作落点。 |
| 时长 | 镜头持续时间 | 通常 6-10 秒，绝对不能超过 10 秒。 |
| 时间轴：台词/旁白 | 声音脚本 | 按 0s-3s、3s-6s 拆分，包含对白、旁白、电话声、环境声。 |
| 转场/情绪承接 | 下一个镜头怎么接 | 动作延续、声音延续、时间跳切、物件转场、情绪反差等。 |
| 其他：场景/细节 | 拍摄和生成细节 | 场景、光线、道具、服装、表情、声音、镜头景别。 |
| image2 生成策略 | 图片怎么生成 | A/B 分段、提示词变化、人物一致性、生成失败时要补的细节。 |

## 镜头情绪弧线

1. **触发**：她一个人处理普通但沉重的家务或家庭问题。
2. **习惯**：家人继续依赖她，因为“她一直都会做”。
3. **忽视**：吃饭、电话、小病小痛或临时状况里，她被理所当然对待。
4. **临界点**：医院、身体不适、一张账单，或一句话，让长期付出的代价变得可见。
5. **回到家**：她继续日常，但行为里出现细微变化。
6. **对话/点醒**：家人、朋友或她自己说出问题的核心。
7. **边界**：她停止承担某个惯性任务。
8. **重新分担**：家人开始笨拙但认真地分担。
9. **释放**：她拥有一个属于自己的安静时刻，最后一句话落下来。

短视频不一定要把九步全部写满。重点是情绪推进清楚：负担 -> 看见 -> 边界 -> 释放。

## 岳姐人设和出镜规则

- 岳姐是账号核心人设，不是故事里的普通角色。
- 默认至少一个镜头要直接出现岳姐本人。
- 关键情绪节点要和岳姐有关：疲惫、停顿、被看见、说出边界、最后释放。
- 家人镜头不能抢主角。家人可以出现，但作用是反衬岳姐、推动岳姐变化、让观众更理解岳姐。
- 如果某个镜头没有岳姐本人，也要和她有关：她的手、背影、围裙、杯子、手机、药袋、饭碗、空椅子、收据等。
- 用户说“打造岳姐人设”“岳姐出现不能少”“可以增加镜头”时，先确保至少一个镜头出现岳姐本人。必要时增加镜头，不要只在旁白里提岳姐。

## 时长规则

- 30-45 秒：5-6 个镜头，每个 5-8 秒。
- 60-75 秒：8-9 个镜头，每个 7-10 秒。
- 90 秒：9-10 个镜头，每个 8-10 秒；如内容装不下，增加镜头数或压缩台词，不允许单镜头超过 10 秒。
- Grok 限制：单个镜头不能超过 10 秒。任何 11 秒、12 秒镜头都必须拆分或重写。
- 台词要能自然说出口。中文旁白平稳语速下，大约每秒 4-5 个字；情绪停顿多时要更少。
- 每一行都要有声音设计，哪怕只是环境声加一句旁白。
