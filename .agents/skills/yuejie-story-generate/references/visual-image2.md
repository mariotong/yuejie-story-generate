# 视觉方向和 image2 提示词

## 明亮中老年视觉规则

这个故事版风格是专门给中老年观众在手机上看的。优先保证清楚、明亮、亲切，再考虑电影感。

- image2 提示词默认加入“明亮清楚，适合中老年观看”。
- 人脸必须看得见，表情必须看得清。避免逆光剪影、阴暗角落、强阴影、深蓝黑夜景、过暗的医院走廊。
- 厨房、餐厅、客厅要干净、温暖、日常：窗边日光、柔和顶灯、浅木色、浅色墙面、桌面细节清楚。
- 医院场景可以偏冷静严肃，但仍要明亮可读：白天光、床品清楚、手、手机、药袋等细节可见。
- 夜晚阳台要用室内暖光或城市光照亮脸，不能把人物做成黑色轮廓。
- 避免灰脏调色、低对比、过多胶片颗粒、明显虚焦、把重要道具藏进浅景深里。
- 如果提示词里出现“昏暗”，必须同时写“但脸部仍然清楚可见”，除非用户明确要黑暗情绪。

## Codex image2 提示词公式

image2 提示词要像制作指令，不要像散文。默认使用中文提示词：

```text
9:16竖屏，真实纪实照片风格，明亮清楚，适合中老年观看，中国女性，约55-60岁，短微卷黑发夹杂少量白发，穿简洁米色针织衫和深色裤子，{场景}，{动作}，{表情}，{道具}，柔和明亮的自然光，脸部均匀受光，构图清楚易读，中景，自然皮肤纹理，有真实生活痕迹的家庭环境，温暖真实色彩，无文字，无水印
```

每个故事版先定义一次主角一致性：

```text
人物一致性：55-60岁中国女性，微胖或普通身材，短卷发夹杂少量白发，素色针织衫/围裙，脸上有疲惫但温和的纹理，不精修，不过度年轻化。
```

后续每个镜头都复用这套人物设定，只改变场景、动作、表情、道具和景别。

## 首帧和尾帧

首帧提示词负责建立镜头开始动作，例如：

- 她看着医院缴费单。
- 她一边煲汤一边接电话。
- 她一个人坐在餐桌旁。
- 她拎着菜进门。
- 她站在病床边。

尾帧提示词负责表现情绪变化，例如：

- 她慢慢放下手机。
- 她停下搅汤的手，望向一边。
- 家人进入画面。
- 她转向阳台的光。
- 桌面上不再只有她一个人的负担。

不要让首帧和尾帧完全一样。即使只是生成两张静态图，观众也要能感到一个镜头里的动作和情绪变化。

## 场景细节清单

- 场景：厨房、餐厅、医院、阳台、客厅、门口、小区路。
- 光线：明亮日光、温暖厨房灯、冷但清楚的医院白光、脸部可见的夜晚阳台光、柔和明亮的台灯。
- 道具：手机、收据、药袋、菜袋、围裙、碗、筷子、保温杯、毛巾。
- 细节：手部动作、皱纹、略乱的台面、吃了一半的饭、未读消息。
- 声音：手机震动、汤锅声、碗筷声、走廊回声、椅子摩擦声、轻微城市声。

## image2 生成策略栏

```text
拆成2段：
A段（5s）：{首帧提示词或主要画面}
B段（4s）：{尾帧提示词或情绪变化}
补镜：{物件特写/没有可读文字的手机屏幕/手部细节}
```

如果生成静态故事版，写“每个镜头生成首帧+尾帧”。如果生成视频素材，写“每个镜头至少首尾两张，关键情绪点补一张近景”。
